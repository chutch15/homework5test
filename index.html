Minecraft Tools & Armor — Custom Scatterplot Museum Exhibit

This visualization places Minecraft items on a 2D map using:
• X-axis: Year of introduction
• Y-axis: Durability (higher = lasts longer)
• Red Squares: Weapons
• Green Circles: Tools
• Blue Chestplates: Armor

This is a Custom Visualization using modded glyph shapes and multiple visual encodings beyond a standard chart.
